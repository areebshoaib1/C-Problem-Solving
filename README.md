# C Problem Solving

This repository contains C programming problems that I solve while learning Programming Fundamentals.

The goal of this repository is to practice problem solving and  programming logic using C.


## Problems

## Arrays
1. Check whether a number is even or odd
2. Check whether a number is positive, negative, or zero
3. Find the largest of 3 numbers
4. Check whether a year is a leap year
5. Check whether a number is divisible by 5 and 11
6. Check whether a character is vowel or consonant
7. Calculate grade of student based on marks
8. Check whether a number is prime

## Arrays
